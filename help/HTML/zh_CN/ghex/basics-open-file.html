Open a file

You may open a file in GHex in the following ways:

  • Click the Open button on the header bar.

  • Press Ctrl+O.

When you open a file, GHex will not automatically turn Insert Mode on. This is so that data can be immediately edited in place as opposed to inserting data, which will be what most users want to do when first opening a pre-existing file.
