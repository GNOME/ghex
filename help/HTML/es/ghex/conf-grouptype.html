Hex data grouping

GHex allows data in the hex pane to be grouped in the following ways:

  • Bytes (8-bit) this is the default

  • Words (16-bit)

  • Longwords (32-bit)

  • Quadwords (64-bit)

This setting may be changed in two ways:

  • From the Group Data As submenu of the Main menu.

  • From the Preferencias dialog.
