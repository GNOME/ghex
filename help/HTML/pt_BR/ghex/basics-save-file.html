Saving files

To save a file in place, maintaining its filename and location, use the Save operation by clicking the Save button on the header bar, or by pressing Ctrl+s.

To save a file to a filename of your choice, use the Save As operation by clicking Save As in the Main menu, or by pressing Shift+Ctrl+S.
